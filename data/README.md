# Data files

`data/processed/online_shoppers_clean.csv` is the cleaned Online Shoppers dataset generated by `scripts/run_pipeline.py` and used by the reproducibility pipeline.

Raw source files are not redistributed here. To rerun the Online Retail II external validation, download the raw `online_retail_II.xlsx` workbook from the UCI Machine Learning Repository and place it at `data/raw/online_retail_ii.xlsx` before running `scripts/run_online_retail_ii_external_validation.py`.
